1、安装 Node.js

首先了解一下什么是 Node.js:

Node.js 是一个基于 Chrome V8 引擎的 Javascript 运行环境

点击 Node.js下载
安装时选项全部默认，一路 Next

检验安装是否成功:
Win+R -> cmd, 在命令窗中输入node -v 和npm -v，出现版本号后，则安装成功

2、安装 Git

点击 Git下载 Git。具体百度搜索。

检验安装是否成功:
Win+R -> cmd, 在命令窗中输入 git –version，出现版本号后，则安装成功
